
# ASMOGRAPHIC — Site local de démo

## Démarrer
1. Télécharge le fichier `asmographic_site.zip` ci‑dessous.
2. Décompresse l’archive.
3. Ouvre `index.html` dans ton navigateur (double‑clic).

Pas de backend, pas de paiement — tout est local pour tester.

## Structure
- `index.html` — page unique (accueil, boutique, lookbook, à propos, contact)
- `style.css` — style sombre, vibe tattoo
- `app.js` — interactions (panier local, newsletter factice)
- `assets/` — logos & images produits (SVG), favicon

## Personnaliser
- Remplace les fichiers `assets/product_*.svg` par de vraies photos.
- Change les textes/prix dans la section "Boutique" (dans `index.html`).
- Mets tes vrais liens Instagram/TikTok/Email.
